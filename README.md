# dokotsubu-app-python

## 📝 アプリ概要
Python（Flask）で作成した簡易SNSアプリです。  
ログイン機能、投稿機能、一覧表示機能を実装しています。

---

## 💡 制作背景
医療事務として電子カルテ導入に関わる中でITに興味を持ち、  
業務効率化や現場改善に貢献できるシステムを作りたいと考え、本アプリを制作しました。

---

## ⚙️ 使用技術
- Python
- Flask
- HTML
